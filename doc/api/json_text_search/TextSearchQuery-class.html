TextSearchQuery

TextSearchQuery class Null safety

A object model class that enumerates the properties of a text search query:

  • phrase is the unmodified search phrase, including all modifiers and tokens;
  • terms is the ordered list of all terms extracted from the phrase used to look up results in an inverted index;
  • mustInclude is a set of unique terms that must appear at least once in a document for it to be included in the search results; and
  • excludeTerms is a set of unique terms that must not appear in a document for it to be included in the search results.
