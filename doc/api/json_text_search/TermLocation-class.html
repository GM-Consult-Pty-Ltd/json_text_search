TermLocation

TermLocation class Null safety

An object describing the location of a term in a JSON document.

  • term is the term/word associated with the TermLocation.
  • the field is the name of the field in which the term occurs;
  • priority is the relevance priority of the field, with 0 having highest priority; and
  • the position is the zero-based location of the start of the term from the start of the text in the field.
