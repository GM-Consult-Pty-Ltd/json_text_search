Hydrates a basic implementation of JsonIndexBuilder with:

  • (required) indexedFields is the names of the indexed fields in the collection, mapped to the relative priority of each field (0 is lowest priority);
  • (optional) tokenizer is a callback that returns a collection of Tokens from text; and
  • (optional) relevanceBuilder is a callback that calculates relevance from a collection of TermLocation.

The default tokenizer Uses a TextAnalyzer instance with English as TextAnalyzerConfiguration and the Porter2Stemmer as TokenFilter.
